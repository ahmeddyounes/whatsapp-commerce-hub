M00-06 Implementation Summary
============================

Files Created:
--------------
1. includes/class-wch-queue.php (175 lines)
   - Manages background job queue using Action Scheduler
   - Registers action hooks for all job types
   - Schedules hourly cart cleanup job
   
2. includes/class-wch-job-dispatcher.php (311 lines)
   - Dispatches single and batch jobs
   - Manages job lifecycle (schedule, cancel, check status)
   - Retrieves pending/failed jobs
   - Provides retry functionality
   
3. includes/class-wch-sync-job-handler.php (316 lines)
   - Handles product/order/inventory sync jobs
   - Implements retry logic with exponential backoff (60s/300s/900s)
   - Max 3 retry attempts
   - Stores job results in transients
   
4. includes/class-wch-broadcast-job-handler.php (274 lines)
   - Processes broadcast campaigns in batches of 50
   - 1 second delay between batches for rate limiting
   - Tracks campaign progress and results
   
5. includes/class-wch-cart-cleanup-handler.php (223 lines)
   - Cleans up carts older than 72 hours
   - Runs hourly via scheduled action
   - Provides cart statistics and manual trigger
   
6. includes/class-wch-abandoned-cart-handler.php (317 lines)
   - Sends reminder messages for idle carts
   - Respects configured delay hours from settings
   - Generates customizable reminder messages
   
7. includes/class-wch-admin-jobs.php (305 lines)
   - Admin UI for job monitoring
   - Shows pending jobs by type
   - Lists failed jobs with retry buttons
   - Cart cleanup statistics and manual trigger
   
8. assets/css/admin-jobs.css
   - Styling for admin jobs UI
   
9. assets/js/admin-jobs.js
   - Admin jobs page interactions
   - Auto-refresh every 30 seconds
